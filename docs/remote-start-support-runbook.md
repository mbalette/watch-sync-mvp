# Remote Start support runbook

Use this copy with users. Do not promise automatic playback. Manual Play is always available.

## Helper unavailable
"Remote Start is paused on your setup right now. Please keep the Watch Sync countdown open and press Play manually when everyone reaches GO."

## Roku not found
"Make sure your phone or laptop and Roku are on the same home Wi‑Fi, then wake the TV and try again. If it still does not show up, use Manual Play tonight."

## Roku mobile control disabled
"On the Roku, enable mobile app control, then try Test Play again. If you cannot find that setting, Manual Play works tonight."

## Same Wi‑Fi
"Remote Start can only try the helper command when this device and the TV are on the same home Wi‑Fi. If they are not, use Manual Play."

## Guest, hotel, or apartment Wi‑Fi
"Some shared Wi‑Fi blocks devices from seeing each other. If Test Play fails there, use Manual Play."

## Test Play failed
"Open the title yourself, pause it on the video, then try Test Play once more. If it still fails, use Manual Play."

## Video already playing
"Pause the title before Test Play. Remote Start is only for paused video."

## Wrong screen or overlay
"Close menus, popups, profile screens, and overlays. Pause on the actual video, then try again."

## Profile picker
"Choose the profile yourself, open the title, pause on the video, then try Test Play."

## Ad
"Wait until the ad finishes, pause on the video, then use Test Play."

## Screensaver or asleep
"Wake the TV, open the title, pause the video, then try again."

## Manual fallback
"Manual Play is the safe fallback: keep the countdown open and press Play manually at GO."

## Beta paused
"We paused the Remote Start beta for safety. Manual Play is still available."
