Updated: 2026-05-05

Truth boundary: Remote Start is local remote-style Play at countdown GO after the user manually opens the title and pauses at the sync point. It is not provider catalog integration, exact timestamp sync, app launch automation, or native streaming-app account/content control. Manual countdown always remains available.

## Safe to build now without hardware

- Roku / Roku TV ECP Play-only adapter.
- LG webOS local pairing + Play/Pause adapter.
- Samsung Tizen beta local key adapter.
- Sony Bravia IP/IRCC Play adapter.
- Fire OS / Android TV / Google TV ADB guided setup beta.

These are buildable as beta/helper paths only. Hardware validation is still required before public `supported` claims.

## Claims blocked on hardware validation

- Real TV pairing acceptance and persistence after helper restart, TV sleep/wake, and TV reboot.
- DHCP/IP-change resilience.
- Netflix/Hulu/Disney+/Prime/Max/YouTube acceptance of Play while paused.
- Normal-user setup completion without support.
- Model/firmware variance for Roku/LG/Samsung/Sony/Vizio.
- Whether Pause is discrete/safe per real device/app combination.
